# Learning GoLang

Tutorial: [Get started with Go](https://go.dev/doc/tutorial/getting-started)

### Running

To run the code:

1. [Download and install](https://go.dev/doc/install) Go
2. Install the package: [rsc.io/quote/v4](https://pkg.go.dev/rsc.io/quote/v4)

```bash
go mod tidy
```

3. Run the code.


```bash
go run .
```
